# Url Shortener

## Tech Stack
* **ejs** - frontend
* **express js** - backend
* **mongodb** - database

## How to run it?
* *git clone https://github.com/dumast/url-shortener.git*
* *cd url-shortener*
* *npm install*
* *npm start*

## How to use it?
* Open *localhost:5000* on your browser
* Enter a url starting with *https://*
* Press *create*
* Copy the link, and voila
